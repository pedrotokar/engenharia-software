# Engenharia de software

Esse repositório contém minhas anotações e exemplos práticos dos conteúdos passados
na matéria de Engenharia de Software, oferecida para o 6º período da graduação em
Ciência de Dados e Inteligência Artificial da FGV EMAp.

Os resumos e atividades encontrados no repositório são todos sobre padrões de
projetos. As pastas estão divididas em grupos de padrões, que foram vistos de
forma conjunta em aula ou que tiveram atividades em comum. Em cada pasta, é
possível encontrar um resumo dos padrões, feito com base no livro *Design
Patterns: Elements of Reusable Object-Oriented Software*, escrito pela conhecida
"Gang of Four". Em algumas pastas, também é possível encontrar o enunciado e
solução de uma atividade envolvendo os padrões da pasta.

Em outras pastas, também é possível encontrar as provas aplicadas na matéria que
eram sobre padrões de projeto, junto com minhas soluções para elas.
